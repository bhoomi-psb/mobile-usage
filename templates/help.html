Frequently Asked Questions (FAQ)

This dataset contains detailed information about user behaviors on mobile devices, including app usage, screen time, device interactions, and location tracking. It is useful for analyzing mobile habits, device preferences, and trends in digital behavior.

The dataset includes information such as:

  • App usage statistics (e.g., app open time, frequency of use, etc.)
  • Screen time data
  • User interaction details (e.g., touch patterns, swipe behavior)
  • Location data (geographical location, activity-related information)

You can access the dataset by requesting it through our data request form on the main website. Once approved, you will be given access to the dataset via a secure download link.

Each dataset is accompanied by a detailed documentation guide, explaining the variables, data types, and how to interpret them. You can find this documentation on the 'Documentation' section of our website.
